NETVPN_FULL_UNICO_COMPLETO_V2

Paquete completo estilo original:

1) instalar.sh
   Instalador unico autoextraible con protocolos base SVRCODE + NETVPN AUTH LOGIN + API VIP + token manager.

2) COMANDOS_INSTALAR_DESINSTALAR_NETVPN.txt
   Comandos completos de instalacion, verificacion, tokens, logs y revert.

3) COMANDOS_ORIGINAL_SVRCODE.txt
   Comandos que traia el instalador original SVRCODE.

4) APP_NETVPN_V27_UPDATE_REAL5S_VIP_DATOS_ADS_AUTH_LOGIN_GEN_INTACTO.zip
   App estable actual que trabaja con AUTH LOGIN.

Uso rapido VPS limpia:
cd /root
unzip NETVPN_FULL_UNICO_COMPLETO_V2.zip
cd NETVPN_FULL_UNICO_COMPLETO_V2
chmod +x instalar.sh
AUTH_MAIN="api.tvchifladito.store" INSTALL_MODE=full bash instalar.sh

Uso VPS ya instalada:
cd /root
unzip NETVPN_FULL_UNICO_COMPLETO_V2.zip
cd NETVPN_FULL_UNICO_COMPLETO_V2
chmod +x instalar.sh
AUTH_MAIN="api.tvchifladito.store" INSTALL_MODE=auth bash instalar.sh
