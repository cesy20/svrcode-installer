SVRCODE - PAQUETE COMPLETO README VISIBLE

Archivos incluidos:

1. instalar.sh
   Instalador único autoextraíble completo. Puedes subir solo este archivo a GitHub si deseas.

2. README.md
   Archivo principal para GitHub. Aquí aparecen todos los comandos ejecutables directamente en la portada del repositorio.

3. COMANDOS_INSTALAR_DESINSTALAR_SVRCODE.txt
   Copia de todos los comandos en texto plano.

4. instalar_desde_github.sh
   Script opcional para instalación desde GitHub.

5. desinstalar_svrcode.sh
   Desinstalador seguro. No toca OpenSSH/root y guarda respaldo.

Uso recomendado en GitHub:
- Subir instalar.sh
- Subir README.md
- Subir desinstalar_svrcode.sh
- Opcional: subir COMANDOS_INSTALAR_DESINSTALAR_SVRCODE.txt

Repositorio usado en comandos:
https://github.com/cesy20/svrcode-installer
